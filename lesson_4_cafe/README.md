# Проект "Кафе" - 3D игра для детей

Интерактивная 3D сцена кафе с возможностью ходить, собирать монетки, говорить с кассиром и покупать еду.

## Как начать:

1. **Откройте проект в Godot:**
   - Запустите Godot
   - Нажмите **"Import"** (не "New Project")
   - Найдите папку `lesson_4_cafe`
   - Выберите `project.godot`
   - Нажмите **"Import & Edit"**

2. **Настройте действия клавиатуры:**
   - Откройте **Project → Project Settings** (`Alt + Enter`)
   - Перейдите на **Input Map**
   - Добавьте действие `interact` и привяжите клавишу **E**

3. **Создайте сцену:**
   - Смотрите подробную инструкцию в `STEP_BY_STEP.md`

## Основные файлы:

- `cafe.tscn` - главная сцена игры (создается в редакторе)
- `Player.gd` - скрипт игрока (управление WASD)
- `Cashier.gd` - скрипт кассира (диалог)
- `Coin.gd` - скрипт монетки
- `Food.gd` - скрипт еды (покупка)
- `UIManager.gd` - управление интерфейсом
- `Global/Game.gd` - глобальный скрипт (монетки, сообщения)

## Механики игры:

1. **Ходьба** - WASD или стрелки
2. **Сбор монеток** - подходите к монеткам, они собираются автоматически
3. **Диалог с кассиром** - подойдите к кассиру и нажмите E
4. **Покупка еды** - подойдите к еде на прилавке и нажмите E (нужны монетки)

## Для преподавателя:

Этот проект предназначен для обучения детей программированию в Godot за 5 занятий. Все скрипты уже готовы, детям нужно создать сцену в редакторе согласно инструкциям.
