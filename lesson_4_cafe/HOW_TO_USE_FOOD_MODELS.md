# Как использовать разные модели еды (пончики, бургер, торт)

## Шаг 1: Подготовка моделей

У вас уже есть три модели:
- Бургер (Бургер.glb)
- Торт (Торт.glb)
- Пончики (нужно добавить)

## Шаг 2: Настройка еды в сцене

### Для каждого вида еды:

1. **В Godot редакторе:**
   - Откройте сцену `cafe.tscn`
   - Найдите узел **Food** (или **Food2**, **Food3**)

2. **Замените модель:**
   - Выберите узел Food
   - Удалите старый **MeshInstance3D** (с BoxMesh)
   - **Right Click на Food → Add Child Node → Instance Child Scene**
   - Выберите модель (например, `Бургер.glb` для первого, `Торт.glb` для второго)
   - Настройте **Transform → Position** и **Scale** модели

3. **Настройте параметры скрипта:**
   - Выберите узел Food
   - В Inspector найдите параметры скрипта:
     - **Food Name**: установите название (например, "Бургер", "Торт", "Пончики")
     - **Price**: цена (скрипт автоматически назначит 1, 2, 3 по названию)
     - **Food Description**: описание (необязательно)

### Пример настройки:

**Food (Бургер):**
- Model: `Бургер.glb`
- Food Name: "Бургер"
- Price: 1 (автоматически)

**Food2 (Торт):**
- Model: `Торт.glb`
- Food Name: "Торт"
- Price: 3 (автоматически)

**Food3 (Пончики):**
- Model: `Пончики.glb` (или название вашего файла)
- Food Name: "Пончики"
- Price: 2 (автоматически)

## Автоматические цены:

Скрипт автоматически назначает цены:
- **Бургер** = 1 монета
- **Пончики** = 2 монеты
- **Торт** = 3 монеты

Можно изменить цены вручную в Inspector.

## Что отображается:

1. **Ценник над едой:**
   - Название (например, "Бургер")
   - Цена (например, "1 монет")

2. **При покупке:**
   - Сообщение: "✅ Вы купили [Название]! Осталось монет: X"
   - Ценник меняется на "✅ [Название] Куплено!"

## Проверка:

1. Запустите игру
2. Подойдите к еде - должно появиться название и цена
3. Нажмите E - должно появиться сообщение с названием еды
4. После покупки - должно появиться сообщение "Вы купили [Название]!"

## Важно:

- Название в **Food Name** должно совпадать с тем, что вы хотите показать игроку
- Модель можно заменить в любой момент - просто удалите старую и добавьте новую через Instance Child Scene
- Масштаб модели можно настроить в Transform → Scale
