# Быстрый старт: Проект "Кафе"

## Как открыть проект в Godot:

1. Откройте Godot
2. Нажмите **"Import"** (НЕ "New Project")
3. Найдите папку `lesson_4_cafe`
4. Выберите файл `project.godot` в этой папке
5. Нажмите **"Import & Edit"**

## Первые шаги после открытия проекта:

1. Godot автоматически создаст файл `cafe.tscn` если его нет
2. Откройте `cafe.tscn` (двойной клик в FileSystem)
3. Следуйте инструкциям из файла `INSTRUCTIONS.md`

## Структура проекта:

- `Global/Game.gd` - глобальный скрипт для управления монетками
- `Player.gd` - скрипт игрока (управление WASD)
- `Cashier.gd` - скрипт кассира (диалог)
- `Coin.gd` - скрипт монетки
- `Food.gd` - скрипт еды (покупка)
- `UIManager.gd` - управление интерфейсом

## Важно:

Проект уже настроен, скрипты готовы. Вам нужно только создать сцену в редакторе Godot согласно инструкциям.
